An easy on the eyes Xcode theme, ported from Ciapre Sublime Text.

Based on Andro but with more syntax highlighting sweetness.

Installation

[NEW!] Automatically

Now intergrated with the all new Xcode Package Manager Alcatraz. After having it successfully installed, just search for Ciapre -> check the checkmark button -> restart Xcode -> Profit

Manually

  1. git clone https://github.com/vinhnx/Ciapre-Xcode-theme.git

  2. cd Ciapre-Xcode-theme

  3. cp Ciapre.dvtcolortheme 'Ciapre 2.dvtcolortheme' 'Ciapre Blue.dvtcolortheme' ~/Library/Developer/Xcode/UserData/FontAndColorThemes

    Note: you may have to make FontAndColorThemes folder in ~/Library/Developer/Xcode/UserData/ if you haven't changed your color themes, by mkdir ~/Library/Developer/Xcode/UserData/FontAndColorThemes.

  4. Restart Xcode, select your new Ciapre theme in Color settings.

  5. Profit! .
